# シッター予約システム (Sitter Reserve)

## 概要
TimeRexのような、Googleカレンダーと連携可能なシッター予約システムのプロトタイプです。
非エンジニアの方でも使いやすいよう、優しく安心感のあるデザインで構築されています。

## 現在の機能
1. **トップページ**: サービスの特徴を伝えるメインビジュアル
2. **予約カレンダー**: 
   - 週ごとの日付選択
   - 日付に応じた空き時間の表示
   - スマホ対応（レスポンシブデザイン）

## 使い方
このシステムをご自身のパソコンで動かすには、ターミナルで以下のコマンドを入力してください。

```bash
cd sitter-scheduler
npm run dev
```

その後、ブラウザで `http://localhost:3000` にアクセスすると画面が表示されます。

## 次のステップ（Googleカレンダー連携）
現在は「仮のデータ」で予約リストを表示しています。
本当にあなたのGoogleカレンダーと同期させるには、Google Cloud Platform（GCP）での設定が必要です。

1. Google Cloud Consoleでプロジェクトを作成
2. Google Calendar APIを有効化
3. APIキーと認証情報を取得
4. このシステムのプログラム設定ファイルに入力

これにより、あなたのカレンダーの「空き時間」だけを自動で表示できるようになります。
